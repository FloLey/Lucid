# Fonts Directory

This directory contains open-source fonts for use in Lucid carousel generation.

## Recommended Fonts

Add the following open-source fonts (TTF or OTF format):

1. **Inter** - Modern sans-serif, excellent for readability
2. **Roboto** - Clean, versatile sans-serif
3. **Open Sans** - Friendly, neutral sans-serif
4. **Montserrat** - Elegant geometric sans-serif
5. **Lato** - Warm, humanist sans-serif
6. **Oswald** - Bold, condensed sans-serif
7. **Playfair Display** - Elegant serif for headings
8. **Raleway** - Elegant sans-serif
9. **Poppins** - Modern geometric sans-serif
10. **Bebas Neue** - Bold display font

## Font Sources

All fonts can be downloaded from:
- [Google Fonts](https://fonts.google.com/)
- [Font Squirrel](https://www.fontsquirrel.com/)

## File Naming Convention

Name font files to match their family name:
- `Inter-Regular.ttf`
- `Inter-Bold.ttf`
- `Roboto-Regular.ttf`
- `Roboto-Bold.ttf`
- etc.
